# Calculadora de Interés Compuesto

Aplicación construida con **Astro**, **React**, **Recharts** y **TailwindCSS**.

## Características

- Aporte inicial y mensual configurables
- Tasa de interés anual (con slider interactivo)
- Tasa de dividendos anual (con slider interactivo)
- Horizonte de inversión de 1 a 60 años
- Toggle de reinversión de dividendos
- Gráfico de área interactivo con Recharts (Valor total, Capital aportado, Intereses acumulados)
- Tabla detallada año a año (desde Año 0)

## Instalación y uso

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Previsualizar build
npm run preview
```
